Для чего:
Инструмент позволяет держать постоянное соединение и обмениваться информацией с сервером в режиме реального времени, в зависимости от того было ли подключение по web-socket - запоминает клиента и отправляет ему сообщения по этому протоколу(шифрует-дешифрует), если это было обычное сообщение то запоминает этого клиента как обычного
Как использовать
1 В классе \Classes\Commander.php написать сценарий на приходящие запросы, например при получении соббщения "{"user_id":"25"}" отдать json формат данных по пользователю с id 25 из базы данных
	, если пришли данные не в формате json или такого пользоваетля нет - разорвать соединения
2 В файле server.php выставить максимальное количество соединений и номер порта - по умолчанию 1100
3 Запустить из консоли start_server.php или из браузера
4 Отключить сервер вы можете запустив client.php
